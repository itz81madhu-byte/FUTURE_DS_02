CUSTOMER CHURN ANALYSIS
OVERVIEW OF PROJECT:
This project analyzes telecom customer data to understand customer behavior and identify factors affecting customer retention. The dataset was cleaned using Python and visualized using Power BI to generate business insights.
DATASET:
The dataset used is the Telco Customer Churn dataset from Kaggle.
It includes customerid,tenure,monthly charge,contract,depepndent,gendr etc...
TOOLS USED:
Python
Jupyter Notebook
Pandas
Power BI
STEPS:
Downloaded the dataset from Kaggle
Cleaned the data using Python in Jupyter Notebook
Performed basic data analysis
Built an interactive dashboard using Power BI
Key KPIs
Total Customers
Total Revenue
Average Monthly Charges
Average Customer Tenure
INSIGHTS:
1. Customer Overview
2. Total customers: 7043
3. Average monthly charge: 64.76
4. Average tenure: 32 months
Total revenue: 16.06M
Insight:
Customer retention is important since revenue depends on long-term customers.
2. Churn Rate
73% customers stayed
27% customers left
Insight:
About 1 out of 4 customers churn, indicating a moderate churn rate.
3. Contract Type
Month-to-Month contracts have the highest churn
1-year and 2-year contracts have lower churn
Insight:
Long-term contracts help reduce customer churn.
4. Monthly Charges
Customers with higher monthly charges churn more.
Insight:
Pricing plays a role in customer dissatisfaction.
5. Payment Method
Electronic check users churn more
Automatic payment users churn less
Insight:
Auto-payment customers tend to be more loyal.
6. Tenure
New customers churn more
Long-term customers stay longer
Insight:
The first few months are critical for customer retention.
✅ Final Insight:
Churn is mainly influenced by short-term contracts, high monthly charges, and early customer experience.
Conclusion
This project demonstrates how data analysis and visualization can help businesses understand customer behavior and improve retention strategies.
